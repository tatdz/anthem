# Anthem Protocol 

Dynamic risk tranching for liquidation/ADL (Arbitrum Sepolia → HyperEVM)

## Features

- ✅ **Real Arbitrum Data Integration**: Uniswap V3 + Block data
- ✅ **Live κ_t Stress Scoring**: Dynamic tranche allocation, Senior/Junior split
- ✅ **SovereignPool Integration**: 0.3% fee generation
- ✅ **ADL Waste Simulation**: Junior protection showcase (JELLY/ETH/POPCAT loss simulation)
- ✅ **HyerEVM deployment coming**: 100% portable to HyperEVM 

## Tech Stack

- **Frontend**: Next.js 15, TypeScript, Tailwind CSS
- **Web3**: Viem, Wagmi, React Query
- **Contracts**: Live on Arbitrum Sepolia
- **Integrations**: Uniswap V3 + Nitro block data
- **Deployment**: Vercel
